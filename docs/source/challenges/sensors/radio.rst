Aprašymas
===========
Šiame projekte, naudosime „micro:bit“ radiją pranešimų siųntimui iš vieno „micro:bit“ į kitą. Parodysi LED ekrane išsiųstą pranešimų kiekį tarp poros „micro:bit“.
