Aprašymas
===========
Šiame projekte, naudositee „micro:bit“ radiją pranešimų siųntimui iš vieno „micro:bit“ į kitą. Parodysite LED ekrane išsiųstą pranešimų kiekį tarp dviejų „micro:bit“.
